# Shared assets (non-runtime)

| Source | Use |
|--------|-----|
| `legacy/tauri_old/public/` | Cursor SVGs and historical static artwork that may be promoted into shared references |
| `legacy/tauri_old/src-tauri/icons/` | Historical app icons and bundle assets |
| `apps/windows/ChurchPresenter/Assets/` | MSIX / Store logos for the packaged WinUI app |

Keep **licensing** and **attribution** with any third-party artwork. Prefer copying into `shared/assets/` only when both Windows and future macOS apps should share the same canonical files.
