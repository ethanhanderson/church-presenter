# Sources

This bibliography records official Renewed Vision / ProPresenter sources used for this local reference. The docs in this folder synthesize the sources for ChurchPresenter development and intentionally do not copy the articles wholesale.

## Core Output Model

- [ProPresenter All Features](https://renewedvision.com/propresenter/all-features) - Public feature landing page used as the starting feature inventory and pricing/subscription context. The fetched public page is mostly a high-level gateway, so adjacent official feature pages and support material are used for feature detail.
- [ProPresenter main page](https://renewedvision.com/propresenter) - Public overview that identifies seamless live editing, multi-screen management, broadcast-quality outputs, stage display, automations/control, Bibles, Remote, SongSelect, edge blend, alpha key, playlists, multiple screens, themes, industry use cases, and up to eight unique outputs.
- [ProPresenter Media & Slides](https://renewedvision.com/propresenter/media-and-slides) - Product landing page that summarizes slide canvas objects, fills/effects, timers/dynamic text, Reflow, Quick Edit, themes, playlists, quick search, arrangements, video inputs, and broad media support.
- [ProPresenter Multi-screen Management](https://renewedvision.com/propresenter/multi-screen-management) - Public feature page for multiple screens, edge blend, Looks, masks, announcements, props, alpha key, SDI, NDI, and Syphon.
- [ProPresenter Stage Display](https://renewedvision.com/propresenter/stage-display) - Public feature page for flexible stage layouts, timers/clocks, current/next text, stage messages, live screen previews, and operator views.
- [ProPresenter Automations & Control](https://renewedvision.com/propresenter/automations) - Public feature page for slide actions, macros, playback markers, timeline, timecode, calendar scheduling, SongSelect, MultiTracks, Planning Center, Resi, Stream Deck, MIDI/DMX, API, and Network Link.
- [ProPresenter Remote](https://renewedvision.com/propresenter/remote) - Public feature page for mobile/tablet remote control of slides, timers, macros, props, messages, audio playlists, libraries, playlists, collections, stage mode, notes, and custom remote views.
- [ProPresenter Streaming](https://renewedvision.com/propresenter/streaming) - Public page that currently mirrors the main feature overview and reinforces capture/streaming, broadcast, and comparison context.
- [ProPresenter Output Layers](https://support.renewedvision.com/hc/en-us/articles/13634000690323-ProPresenter-Output-Layers) - Defines ProPresenter's fixed output layers, identifies mask/slide/media layer behavior, and explains the important background-color-over-media behavior introduced around ProPresenter 7.11.
- [What's the Difference Between Screens and Outputs?](https://support.renewedvision.com/hc/en-us/articles/360041879993-What-s-the-Difference-Between-Screens-and-Outputs) - Defines Screen as ProPresenter's render target and Output as the hardware/virtual destination. Lists screen types and output types.
- [Screen Configuration in ProPresenter](https://support.renewedvision.com/hc/en-us/articles/360041879173-Screen-Configuration-in-ProPresenter) - Explains audience versus stage screens, the Configure Screens workflow, output options, screen types, hardware tab, placeholders, performance stats, and HDR/bit-depth mention.
- [Using Looks to Show Different Screen Content in ProPresenter](https://support.renewedvision.com/hc/en-us/articles/360041407174-Using-Looks-to-Show-Different-Screen-Content-in-ProPresenter-7) - Defines Looks as layer visibility per audience screen, explains Look Presets, alternate themes, and changing Looks from the UI, slide actions, screens menu, or macros.
- [Understanding the ProPresenter User Interface](https://support.renewedvision.com/hc/en-us/articles/360041345954-Understanding-The-ProPresenter-User-Interface) - Provides operator-surface context: audience/stage toggles, preview window, clear groups, layer clear buttons, transport controls, announcement target indicator, and show controls.

## Stage, Announcements, and Generated Content

- [Using a Stage Screen to its Full Potential](https://support.renewedvision.com/hc/en-us/articles/360041407794-Using-a-Stage-Screen-to-its-Full-Potential) - Defines stage layouts, available stage layout elements, stage screen previews, stage messages, layout assignment, slide actions, Stage Only, and Stage + Audience behavior.
- [Using Slide Destination within ProPresenter](https://support.renewedvision.com/hc/en-us/articles/47985491763219-Using-Slide-Destination-within-ProPresenter) - Explains sending slides to Stage screens only, the Audience + Stage default, the Stage Only transport indicator, and command/menu/action entry points.
- [Create a Multiviewer Output in ProPresenter](https://support.renewedvision.com/hc/en-us/articles/360051396653-Create-a-Multiviewer-Output-in-ProPresenter) - Explains building a stage-screen multiviewer with Screen Preview objects, labels, strokes, layout assignment, and the hardware-performance caveat for additional screens.
- [The Announcement Layer](https://support.renewedvision.com/hc/en-us/articles/360041809953-The-Announcement-Layer) - Explains targeting a presentation to the announcement layer, routing it through Looks, running it simultaneously with the main presentation, and clearing only announcements.
- [What are Show Controls?](https://support.renewedvision.com/hc/en-us/articles/4412263446035-What-are-Show-Controls) - Explains the Show Controls area introduced around ProPresenter 7.5/7.6 for Props, Messages, Timers, Stage, Audio Bin, and Macros.
- [Setting up Timers in ProPresenter 7](https://support.renewedvision.com/hc/en-us/articles/360050782494-Setting-up-Timers-in-ProPresenter-7) - Defines countdown, countdown-to-time, elapsed timers, overrun, reset/start/stop, and multi-timer behavior.
- [How to Create a Countdown for an Audience Screen](https://support.renewedvision.com/hc/en-us/articles/360050786794) - Explains Timer + Theme + Message, linked text, dismiss behavior, transitions, header-triggered timers, and slide-action timers.
- [Using Timers on Stage Screens](https://support.renewedvision.com/hc/en-us/articles/360053250613) - Adds stage timer objects, system clock, video countdown, color triggers, and visibility conditions.
- [Planning Center Live and Planning Center Live Stage Timers](https://support.renewedvision.com/hc/en-us/articles/1500006143281-Planning-Center-Live-and-Planning-Center-Live-Stage-Timers) - Explains Planning Center Live control, stage timer modes, and the distinction between PCO Live advancement and ProPresenter playlist advancement.

## Coverage Notes

- Official `support.renewedvision.com` pages are preferred when available.
- Some article URLs have older or alternate slugs; this bibliography uses canonical accessible pages where possible.
- Current dedicated support articles solely for Props, Messages, Masks, calendar scheduling, and timecode are sparse or were not found in this pass. For those topics, this reference uses current feature pages plus current Remote, Control, timer, clear-group, Show Controls, output-layer, and stage articles.
- The official ProPresenter 7 User Guide PDF is retained only as historical supplemental context in older subsystem notes. The feature docs use current public feature pages and current knowledge-base articles as source of truth.
- The official library-creation article linked from the playlist article requires sign-in when accessed directly. Library behavior is therefore documented from accessible playlist/UI/search sources and should be revisited before detailed library-creation implementation.
